## Описания проектов
Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Специалист по Data Science".
| Название проекта | Описание | Используемые библиотеки |
| :-------------------- | :-------------------- |:--------------------|
| [Промышленность](https://github.com/KateVedrova/yandex-praktikum-projects/tree/master/Promyshlennost) | Чтобы оптимизировать производственные расходы, металлургический комбинат ООО «Так закаляем сталь» решил уменьшить потребление электроэнергии на этапе обработки стали. Предстоит построить модель, которая предскажет температуру стали. | matplotlib, numpy, pandas, seaborn, catboost, sklearn |
| [Восстановление золота из руды](https://github.com/KateVedrova/yandex-praktikum-projects/tree/master/Vosstanovlenie_zolota_iz_rudy) | Необходимо подготовить прототип модели машинного обучения для «Цифры». Компания разрабатывает решения для эффективной работы промышленных предприятий. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды. Нужно использовать данные с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками. | matplotlib, numpy, pandas, sklearn |
